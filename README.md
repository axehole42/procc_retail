# ProcSounds

World of Warcraft addon for Classic and Anniversary Official clients.

This repository contains a lightweight combat-alert addon with:

- combat proc sound cues
- warrior execute-range and execute-hit alerts
- lockpicking and Heavy Junkbox tracking
- movable visuals and saved settings

## Status

This is a personal project and is not packaged or documented as a public release.

## Assets

Custom audio files live in `Sounds/`.

Custom execute cue artwork lives in `Visuals/` and should be provided as `execute.tga` or `execute.blp`.
